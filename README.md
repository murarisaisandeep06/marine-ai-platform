# 🌊 Marine AI Platform

## 📌 Overview
Marine AI Platform is an AI-driven unified data system designed to analyze:
- Oceanographic data
- Fisheries data
- Biodiversity insights

It integrates multiple datasets and provides visualization, analytics, and AI-powered insights.

---

## 🚀 Features
- 🌐 Interactive ocean data visualization
- 🎣 Fisheries analytics and tracking
- 🐟 Biodiversity monitoring
- 🤖 AI chatbot for queries
- 📊 Real-time data processing

---

## 🛠 Tech Stack
- **Frontend:** React (Vite)
- **Backend:** FastAPI
- **AI Service:** Python (Chatbot)
- **DevOps:** Docker, Docker Compose
- **Cloud (optional):** AWS
